# To interact with the GUI
1. Select Load Data file
2. Select the .csv/.xls/.xlsx file you wish to create a profile of
3. Select the directory to save the profile (.html file)

# Required Libraries
- PyQt4
- Pandas
- matplotlib
- pandas_profiling
